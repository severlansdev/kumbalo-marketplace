When you design something, format handoff like this:

```

- Background: #____
- Text: #____
- Accent: #____

Spacing:
- Padding: __px
- Gap: __px
- Margin: __px

Typography:
- Font: ____
- Size: __px
- Weight: ___
- Line-height: __

Effects:
- Shadow: 0 __px __px rgba(__, __, __, __)
- Radius: __px
- Blur: __px

Animation:
- Duration: __ms
- Delay: __ms
- Easing: ____ (e.g., power4.out, elastic.out(1, 0.3))
- Trigger: ____ (hover, scroll, load)
```

This ensures web-developer implements exactly what you envision.
