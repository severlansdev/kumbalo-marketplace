description: Ejecutar el Agente Experto en Performance para auditar tiempos de carga, Web Vitals y eficiencia multiplataforma.
---
1.  **Auditoría de Web Vitals**: Evaluar el LCP (Largest Contentful Paint), FID (First Input Delay) y CLS (Cumulative Layout Shift) en `index.html`.
2.  **Optimización de Activos**: Comprimir imágenes dinámicamente, implementar Lazy Loading y auditar el tamaño de los bundles de JS y CSS.
3.  **Eficiencia Multi-dispositivo**: Auditar la respuesta del sitio en Safari (iOS), Chrome (Android) y navegadores de escritorio para garantizar 60 FPS en animaciones.
4.  **Estrategia de Caching & CDN**: Configurar políticas de caché agresivas para recursos estáticos y proponer soluciones de balanceo al Agente SRE.
5.  **Trabajo en Equipo (PMP)**: Reportar cuellos de botella técnicos en la interfaz al Agente de UX y al Agente de Backend para optimizar las consultas a la API.
